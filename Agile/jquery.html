JQuery

What is JQuery?

Most developers have run into the small differences between the way different browsers interpret client side JavaScript. This can cause major frustration in creating the client side code. JQuery is a JavaScript Library that is meant to ease the writing of cross browser JavaScript code. JQuery is extensible and has many plugins available. There is also an entire set of plugins available for user interface manipulations.

How does it work?

The power of JQuery is DOM manipulation. If you need to change anything in the HTML page, JQuery will allow you to do this in a manner that is consistent between all major browsers. The library also allows you to chain commands together, which also helps to make JQuery more powerful.

It is extremely easy to setup. To include the library in your HTML pages all you need to do is download the JavaScript file from www.jquery.com and include the that file in your HTML

  <html>
  <head>
    <script type="text/javascript" src="jquery.js"></script>
    <script type="text/javascript">
      // Your code goes here
    </script>
  </head>
  <body>
    <a href="http://jquery.com/">jQuery</a>
  </body>
  </html>

Examples

DOM Manipulation

Traditionally JavaScript programmers use the following functions to retrieve HTML elements to alter them in real time:

document.getElementById()
document.getElementsByName()

document.getElementsByTagName()

These functions are fairly limited in their use. What if you required to get a collection of elements that have the same CSS class. You would have to loop through every element and return the collection. JQuery simplifies this because you can select any HTML element by supplying the CSS information.

To select an element by the id and you would use this functionality.

var elements = $('#elementID'); //by id
var elements = $('.elementClass'); //by class

The is good for selecting elements because you can reference them by their CSS selectors, but what about manipulating all of the elements that fit the search.

//Update the class names of every link
$('a').aliClass('newClassName');

The JQuery commands can be chained together. For instance we can ali a new class name and update the color of the links. 

$('a').aliClass('newClassName').css('color','black');

AJAX

AJAX has always had cross browser problems. This is because Internet Explorer has implemented the XMLHTTPRequest object differently from the other browsers. To overcome this developers had to traditionally do something like this.

if (window.XMLHttpRequest)     // Object of the current windows
{ 
    xhr = new XMLHttpRequest();     // Firefox, Safari, ...
} 
else if (window.ActiveXObject)   // ActiveX version
 {
    xhr = new ActiveXObject("Microsoft.XMLHTTP");  // Internet Explorer 
 }

Instead, JQuery offers a function to handle AJAX requests in the same manner across all browsers.

$.ajax({
      url: "script.php",
      global: false,
      type: "POST",
      data: ({id : this.getAttribute('id')}),
      dataType: "html",
      success: function(msg){
         alert(msg);
      }
   }
);

This brings down the response for that php script. The developer can do whatever they want with the results.

UI

JQuery also provides plugins for manipulating the user interface. The plugins provide widgets that can be used and animations for providing users a rich user interface.

The following widgets are provided in the JQuery UI library.

  • Accordion
  • Datepicker
  • Dialog
  • Progressbar
  • Slider
  • Tabs

The following animations are provided by the JQuery UI library.

  • Add Class
  • Remove Class
  • Animate
  • Effect
  • Hide
  • Show
  • Switch Class
  • Toggle
  • Toggle Class
